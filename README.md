# Ruboty::Metal

A Ruboty handler to say metal words.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-metal'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-metal

## Usage

```
@ruboty metal - Say a metal word.
```
